Content Strategy

My site will be a professional portfolio that is one page and includes the following sections.

Welcome: I would like movement on my profile to catch the users eye, so I will have a moving photo/background. My name will be overlaid the photo in big font.Right underneath my name will be icons that connect to my Linkedin, github, and email address so people can contact me. Below the photo there will be open space that states, "I am a web developer".

About Me: This section will have a photo of myself on the left. To the right there is text that explains my career background and what skills I can offer as a Web Developer. It will include programming language as well as soft skills. I will add in some hobbies so the website comes off as personal and the user can get to know me more.

Portfolio:I only have three images at this point. I will include my capstone project and hopefully two other projects that will be volunteer websites. These images will be responsive so that they stack on top of each other in mobile view.
